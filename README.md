# hub-main-cn

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存储在仓库中，可直接通过浏览器访问。

仓库不绑定任何特定域名或网站，仅作为 HTML 文件的集中存储与版本管理平台。每个页面独立存在，互不依赖，适合快速部署或存档。

## 目录说明

```
.
├── index.html          # 首页 / 导航页（可选）
├── pages/              # 独立 HTML 页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有独立 HTML 页面，文件名建议使用英文小写，避免特殊字符。
- 根目录下的 `index.html` 可作为入口页面，用于展示页面列表或导航信息（非必须）。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，包含所需的样式、脚本与结构。
- 页面之间无交叉引用或依赖关系，可单独部署或访问。
- 页面内容由贡献者自行负责，仓库仅作存储与版本管理。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或对已有页面进行改进。
- 提交前请确保页面在主流浏览器中可正常显示，且不包含恶意代码。
- 不建议在页面中嵌入外部不可控资源（如第三方 CDN 脚本），以保证长期可访问性。
- 仓库维护者保留对页面内容进行审核或移除的权利。
